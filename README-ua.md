### Опис завдання

Ваше завдання — створити цей компонент картки попереднього перегляду з 3 колонками та максимально наблизити його до дизайну.

Ви можете використовувати будь-які інструменти, які вам подобаються, щоб виконати завдання. Тож, якщо у вас є щось, що ви хотіли б потренувати, сміливо спробуйте.

Ваші користувачі повинні мати змогу:

- Переглядати оптимальний макет залежно від розміру екрана свого пристрою
- Переглядати стани наведення курсора для інтерактивних елементів

### Використані технології

- HTML5 для розмітки
- CSS3 стилізація
- Flexbox
- Grid
